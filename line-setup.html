LINE 連携 設定ガイド

患者さんへの予約確定・リマインダー通知を LINE で自動送信するための設定手順を、画面キャプチャ付きでご説明します。

所要時間 約 20〜30 分 難易度 ★★☆☆☆ 対象 医院管理者 最終更新 2026年4月
Chapter 0

概要と準備するもの

LINE 連携を設定すると、予約確定・変更・リマインダーが患者さんの LINE に自動で届くようになります。まず必要なものを確認しましょう。

LINE 連携でできること

通知タイミング 内容 対象
予約受付時 「予約を受け付けました。確認後にご連絡します」 患者さん
予約確定時 「予約が確定しました」＋日時・医院名 患者さん
前日リマインダー 「明日〇時にご予約があります」 患者さん
予約変更時 「予約日時が変更されました」 患者さん
暗証番号再発行時 「マイページの暗証番号が再発行されました」 患者さん

必要なもの

  • ✓LINE アカウント（個人のものでOK）— LINE Developers へのログインに使用
  • ✓LINE 公式アカウント（無料プランで可）— 患者さんへのメッセージ送信元
  • ✓SyncotBase の管理者アカウント — 設定を入力するために必要
  • ✓パソコン（スマホではなくPCを推奨）
⚠️ 注意

LINE 公式アカウントの「Messaging API」プランは無料で利用開始できますが、月1,000通を超えると有料になります（1通あたり約3円〜）。患者数が少ない段階は無料範囲内で収まる医院がほとんどです。

設定の全体の流れ

  1. LINE Developers にログイン・プロバイダー作成

    LINE の開発者向けサイトで組織（プロバイダー）を作ります。

  2. Messaging API を有効化してプロバイダーに紐づける

    LINE Official Account Manager の設定から Messaging API をオンにし、作成したプロバイダーと連携させます。

  3. Channel Secret / Access Token を取得

    SyncotBase との連携に必要な認証キーを発行します。

  4. Webhook URL を LINE に設定

    SyncotBase の URL を LINE 側に登録します。

  5. SyncotBase 管理画面に入力して完了

    取得したキーを SyncotBase に登録します。

Step 1

LINE Developers にログイン

LINE Developers は、LINE のビジネス・開発機能を管理するサイトです。お持ちの LINE アカウントでログインできます。

ログイン手順

  1. LINE Developers を開く

    ブラウザで developers.line.biz にアクセスします。

  2. 「ログイン」をクリック

    右上の「ログイン」ボタンから LINE アカウントでログインします。QRコードをスマホで読み取る方法が簡単です。

  3. 開発者アカウントの登録

    初回ログイン時は名前・メールアドレスの入力が必要です。医院名または担当者名を入力してください。

Step 2

プロバイダーを作成する

プロバイダーは「送信者の組織」を表します。医院名で作成しておくと管理しやすくなります。

💡 プロバイダーとは？

LINE でいう「会社・組織」の単位です。1つのプロバイダーの中に複数のチャンネルを作れます。医院名で1つ作れば十分です。

  1. コンソール画面で「新規プロバイダー作成」

    ログイン後のコンソール画面で「プロバイダーを作成する」ボタンをクリックします。

  2. プロバイダー名を入力

    医院名（例：「○○整骨院」）を入力して「作成」をクリックします。

Step 3

Messaging API を有効化してプロバイダーに紐づける

LINE Developers のプロバイダーからチャンネル作成を開始し、LINE Official Account Manager で既存の公式アカウントに Messaging API を有効化します。

3-1. LINE Developers からチャンネル作成を開始

  1. Step 2 で作成したプロバイダーを開く

    コンソール左メニューからプロバイダー名をクリックし、「チャンネル設定」タブを表示します。

  2. 「Messaging API」アイコンをクリック

    「チャンネルを作成するには、チャンネルの種類を選択します」という画面で Messaging API のアイコンをクリックします。

  3. 「LINE公式アカウントを作成する」ボタンをクリック

    「LINE DevelopersコンソールからMessaging APIチャンネルを直接作成することはできなくなりました」という画面が表示されます。緑色の 「LINE公式アカウントを作成する」 ボタンをクリックします。
    ※ すでに公式アカウントをお持ちの場合も、このボタンから LINE Official Account Manager へ移動します。

3-2. LINE Official Account Manager で Messaging API を有効化

ボタンをクリックすると account.line.biz（LINE Official Account Manager）が開きます。以下の手順で既存の公式アカウントに Messaging API を有効化します。

  1. 自院の公式アカウントを選択

    ホーム画面にアカウント一覧が表示されます。連携したい医院のアカウントをクリックして開きます。

  2. 左メニューの「設定」をクリック

    画面左下または左メニューにある「設定（歯車アイコン）」をクリックします。

  3. 「Messaging API」メニューを開く

    設定メニューの一覧から「Messaging API」をクリックします。

  4. 「Messaging APIを利用する」ボタンをクリック

    Messaging API の説明画面が表示されます。「Messaging APIを利用する」ボタンをクリックします。

  5. プロバイダーを選択する

    「プロバイダーを選択してください」のドロップダウンから、Step 2 で作成したプロバイダー（医院名）を選びます。

  6. 「同意する」→「OK」で確定

    利用規約への同意確認と、確認ダイアログが表示されます。内容を確認して進めます。一度紐づけると後から変更できないため、プロバイダー名を必ず確認してください。

  7. LINE Developers でチャンネルを確認

    LINE Developers（developers.line.biz）のコンソールに戻り、プロバイダーの中に Messaging API チャンネルが追加されていれば完了です。

🔴 注意：プロバイダーの紐づけは変更不可

一度プロバイダーに紐づけると後から変更できません。Step 2 で作成した正しいプロバイダー名になっているか、必ず確認してから「OK」を押してください。

Step 4

Channel Secret と Access Token を取得する

SyncotBase と LINE を繋ぐための「認証キー」を2種類取得します。この手順が最も重要です。

4-1. Channel Secret（チャンネルシークレット）の確認

Channel Secret はチャンネル作成後に自動で発行されています。

  1. 「チャンネル基本設定」タブを開く

    チャンネル設定画面の上部にあるタブから「チャンネル基本設定」をクリックします。

  2. 「Channel secret」欄を探す

    ページを少し下にスクロールすると「Channel secret」という項目があります。

  3. 値をコピーしてメモしておく

    右側の「コピー」ボタンを押してコピーします。テキストエディタ等に貼り付けて保管してください。

4-2. Channel Access Token（アクセストークン）の発行

Access Token は手動で「発行」操作が必要です。

  1. 「Messaging API 設定」タブを開く

    画面上部のタブから「Messaging API 設定」をクリックします（「チャンネル基本設定」の隣にあります）。

  2. ページ下部の「チャンネルアクセストークン」欄を探す

    ページをスクロールして「チャンネルアクセストークン（長期）」という欄を探します。

  3. 「発行」ボタンをクリック

    「発行」ボタンを押すと長い文字列が表示されます。これがアクセストークンです。

  4. 値をコピーしてメモしておく

    右側のコピーボタンでコピーし、テキストエディタに保管します。このトークンは再表示できないため、必ず保管してください。

🔴 重要：トークンの取り扱い

Channel Secret と Access Token は 絶対に第三者に教えないでください。知られると悪用される可能性があります。メールへの貼り付けや SNS への投稿は厳禁です。

スクリーンショット 4-B
「Messaging API 設定」タブ
チャンネル基本設定の隣にあるタブをクリックします
📋 この時点で手元に用意できているもの

✅ Channel secret（チャンネルシークレット）
✅ Channel access token（チャンネルアクセストークン）

Step 5

Webhook URL を LINE に設定する

患者さんが友だち追加したり、マイページ連携する際のイベントを SyncotBase が受け取るための URL を設定します。

Webhook URL の確認（SyncotBase 管理画面）

まず、SyncotBase 側で発行されている Webhook URL を確認します。

  1. SyncotBase 管理画面にログイン

    システム管理者から共有された URL（例：syncotbase.jp/admin）にアクセスします。

  2. 「テナント一覧」→ 医院名をクリック

    自院の名前をクリックして詳細ページを開きます。

  3. 「LINE 連携設定」欄の Webhook URL をコピー

    「Webhook URL」の右にある「コピー」ボタンを押します。

Webhook URL の形式
https://syncotbase.jp/api/webhook/line/{医院のID}

LINE Developers に Webhook URL を貼り付ける

  1. LINE Developers の「Messaging API 設定」タブへ戻る

    ブラウザで LINE Developers を開き、先ほどのチャンネルを選択します。

  2. 「Webhook URL」欄に貼り付ける

    「Webhook URL」の入力欄に、コピーした URL を貼り付けます。

  3. 「更新」ボタンをクリック

    入力欄の右にある「更新」ボタンを押して保存します。

  4. 「Webhookの利用」を ON にする

    Webhook URL 欄のすぐ下に「Webhookの利用」というトグルがあります。必ず ON にしてください。

  5. 「検証」ボタンで疎通確認

    「検証」ボタンを押して「成功」と表示されれば接続完了です。

⚠️ 「応答メッセージ」は OFF にする

Messaging API 設定の「応答設定」欄にある「応答メッセージ」と「あいさつメッセージ」は OFF にしておくことを推奨します。ONのままだと、患者さんが友だち追加した際にデフォルトのメッセージが自動返信されてしまいます。

Step 6

SyncotBase 管理画面にキーを入力する

手順4で取得した2つのキーを SyncotBase に登録します。

  1. SyncotBase 管理画面で自院の詳細ページを開く

    「テナント一覧」から自院の「編集」ボタンをクリックします。

  2. 「LINE 連携設定」欄を探す

    ページをスクロールすると「LINE 連携設定」というセクションが表示されます。

  3. 「Channel Secret」を入力

    手順4-1でコピーした Channel secret を貼り付けます。👁️ アイコンで表示・非表示を切り替えられます。

  4. 「Channel Access Token」を入力

    手順4-2でコピーしたアクセストークンを貼り付けます。長い文字列ですが全部貼り付けてください。

  5. 「保存する」ボタンをクリック

    ページ下部の「保存する」ボタンを押します。「保存しました」と表示されれば完了です。

Step 7

友だち追加 URL を設定する（推奨）

患者さんが簡単に友だち追加できるよう、LINE 公式アカウントの友だち追加 URL を登録しておきましょう。これにより予約完了画面に「友だち追加」ボタンが自動表示されます。

友だち追加 URL の確認方法

  1. LINE Official Account Manager にログイン

    account.line.biz にアクセスして、先ほど作成したアカウントを開きます。

  2. 「設定」→「アカウント情報」を開く

    左メニューの「設定」からアカウント情報を確認します。

  3. 「基本 ID」を確認

    「@」で始まる英数字の ID が表示されています（例：@abc12345）。

  4. 友だち追加 URL を作成

    基本 ID を使って以下の形式の URL を作ります。

友だち追加 URL の形式
https://line.me/R/ti/p/@abc12345

SyncotBase への登録

  1. SyncotBase の LINE 連携設定欄を開く
  2. 「友だち追加URL」欄に貼り付け

    作成した URL を「友だち追加URL（任意）」欄に貼り付けます。

  3. 「保存する」をクリック
💡 友だち追加 URL の効果

この URL を設定すると、患者さんがネット予約を完了した画面に「LINE で友だち追加する」ボタンが自動的に表示されます。友だち追加してもらうことで、以後の通知が LINE に届くようになります。

Step 8

動作確認をする

すべての設定が完了したら、実際に通知が届くか確認しましょう。

確認手順

  1. LINE 公式アカウントを友だち追加する

    設定した友だち追加 URL から、自分のスマホの LINE で友だち追加します。

  2. SyncotBase で患者情報に LINE ID を紐づける

    患者さんがマイページの「LINE 連携」機能から連携するか、管理者が患者詳細画面から LINE ユーザー ID を設定します。

  3. テスト予約を作成して承認する

    患者詳細ページから予約を作成し、「確定」ボタンを押します。

  4. LINE にメッセージが届いたか確認

    スマホの LINE アプリを開き、SyncotBase からの通知メッセージが届いていれば設定完了です 🎉

よくある質問

FAQ

設定中によくある疑問をまとめました。

Q. LINE 公式アカウントは有料ですか？

月1,000通まで無料です。月の予約件数が少ない医院なら無料範囲内に収まります。1,000通を超える場合は「ライトプラン（月5,000円〜）」などが必要になります。詳しくは LINE 公式の料金ページをご確認ください。

Q. Channel Secret と Access Token はどこかに保管すべきですか？

SyncotBase に入力した後は、システム上で安全に管理されます。ただし、トークンを再発行した場合は新しい値に更新する必要があるため、安全な場所（パスワードマネージャーなど）に保管しておくことを推奨します。

Q. 通知を LINE ではなくメールにしたいです

SyncotBase の設定画面でメール通知の ON/OFF を切り替えられます。LINE・メールの両方を同時に有効にすることも可能です。

Q. 複数の院で LINE 連携したい

院ごとに別のチャンネルを作成し、それぞれの Channel Secret / Access Token を各院の SyncotBase 設定に入力してください。同じプロバイダーの下に複数チャンネルを作成できます。

Q. LINE の友だちになっていない患者さんには届きますか？

届きません。LINE 通知を受け取るには、患者さんが LINE 公式アカウントを「友だち追加」し、かつマイページで LINE 連携をしている必要があります。患者さんに友だち追加を案内するため、受付でのQRコード掲示や、LINE 連携 URL の共有をお勧めします。

Q. Access Token を間違えて削除してしまいました

LINE Developers の「Messaging API 設定」タブで「再発行」ボタンから新しいトークンを発行できます。発行後、SyncotBase の管理画面で新しい値に更新してください。古いトークンは即時無効になります。

考えられる原因対処法
