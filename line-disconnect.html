LINE連携 解除ガイド

SyncotBase と LINE Official Account の連携を安全に解除する手順を説明します。

所要時間：約 5 分 対象：院長・管理者 難易度：かんたん
Chapter 0

解除前に確認しておくこと

解除を実行する前に、以下の影響範囲を必ずご確認ください。

解除後に起きること

項目 解除前（連携中） 解除後
予約確定通知 ✔ LINEで送信 ✘ 送信されない
リマインダー通知 ✔ LINEで送信 ✘ 送信されない
キャンセル通知 ✔ LINEで送信 ✘ 送信されない
メール通知 設定に依存 変わらず送信
患者・予約データ — ✔ 削除されない
LINE公式アカウント — ✔ そのまま存在
患者の友だち登録 — ✔ 解除されない
⚠️ 注意

解除後は即座に LINE 通知が停止します。患者への通知手段がメールのみになるため、解除の前にメール通知が有効になっているか確認してください。

✔ 安心ポイント
  • 予約・患者データは一切削除されません
  • 解除後も再連携できます（LINE_SETUP_GUIDE を参照）
  • 患者が LINE で友だち追加したままの状態は変わりません
Step 1 — SyncotBase

設定画面を開く

SyncotBase の管理画面にログインし、設定ページの LINE タブを開きます。

  1. 管理画面にログインする

    SyncotBase の管理画面（例：https://syncotbase.com/あなたの医院ID/settings）にアクセスしてください。

  2. 左サイドバーの「設定」をクリック

    歯車アイコンの「設定」メニューを選択します。

  3. 「LINE連携」タブを選択する

    設定ページ上部のタブ一覧から「LINE連携」を選択してください。

スクリーンショット 1
設定ページ — LINE連携タブ
Channel Secret・Access Token が入力済みの状態
💡 ヒント

LINE連携タブを開くと、現在設定されている Channel Secret・Access Token・友だち追加URL が表示されています（Channel Secret は伏せ字表示）。これらが表示されていれば連携中の状態です。

Step 2 — SyncotBase

「LINE連携を解除する」ボタンを押す

フォーム下部の赤いボタンをクリックします。

  1. ページ下部の赤いボタンを確認する

    LINE連携設定フォームの一番下に 「LINE連携を解除する」 というボタンが表示されています。このボタンは 連携が設定済みのときのみ 表示されます。

  2. ボタンをクリックする

    クリックすると確認ダイアログが表示されます。この時点ではまだ解除されません。

スクリーンショット 2
「LINE連携を解除する」ボタン（赤色・左下）
右側には「保存する」ボタンが並んで表示される
⚠️ 注意

「LINE連携を解除する」ボタンと「保存する」ボタンは別々の操作です。「保存する」を押しても解除はされません。

Step 3 — SyncotBase

確認ダイアログで「解除する」を押す

確認ダイアログで内容を確認し、解除を実行します。

  1. 確認ダイアログの内容を確認する

    「Channel Secret・Access Token・友だち追加URLをすべて削除します。解除後はLINE通知が送信されなくなります。」と表示されます。

  2. 「解除する」ボタンをクリックする

    赤いボタン「解除する」をクリックしてください。キャンセルしたい場合は「キャンセル」を押します。

  3. 「LINE連携を解除しました」と表示されることを確認

    画面上部に緑色の成功メッセージが表示されれば解除完了です。Channel Secret・Access Token フィールドが空になっていることを確認してください。

スクリーンショット 3
確認ダイアログ
「解除する」（赤）と「キャンセル」の2つのボタン
スクリーンショット 4
解除完了後の画面
「LINE連携を解除しました」の緑色メッセージ＋フィールドが空
✔ ここまでで SyncotBase 側の解除は完了です

Step 4・5 は LINE 側の設定を整理したい場合のみ実施してください。セキュリティ上はなるべく実施することをおすすめします。

Step 4 — LINE側（推奨）

LINE Developers で Webhook を無効化する

SyncotBase 側の解除だけでも通知は止まりますが、LINE Developers コンソールでも Webhook を無効化しておくと、不要なリクエストが LINE から届かなくなります。

  1. LINE Developers コンソールにアクセスする

    https://developers.line.biz/ を開き、LINE アカウントでログインします。

  2. 対象の Channel を選択する

    Providers 一覧から、SyncotBase と連携していた Messaging API Channel を選択してください。

  3. 「Messaging API」タブを開く

    Channel 詳細ページ上部のタブから「Messaging API」を選択します。

  4. 「Webhook settings」の「Use webhook」をオフにする

    「Webhook settings」セクションの「Use webhook」トグルを オフ（グレー） にしてください。

スクリーンショット 5
LINE Developers — Webhook settings
「Use webhook」をオフにした状態
💡 補足

Webhook URL はそのまま残しておいても問題ありません。将来再連携するときに再利用できます。

Step 5 — LINE側（セキュリティ強化・任意）

Channel Access Token を失効させる

SyncotBase から削除されたトークンをさらに LINE 側でも無効化することで、万一トークンが漏洩していた場合でも悪用されなくなります。

  1. 「Messaging API」タブの「Channel access token」セクションを開く

    Step 4 で開いた「Messaging API」タブの下部にあります。

  2. 「Reissue」ボタンをクリックする

    新しい Access Token が発行され、古いトークン（SyncotBase に設定していたもの）は 30日後に自動的に失効 します。即時失効させたい場合は「Revoke」ボタンを使います。

スクリーンショット 6
LINE Developers — Channel access token セクション
「Reissue」または「Revoke」ボタンの位置
⚠️ 注意

「Revoke」（即時失効）を行うと他のシステムとの連携も即座に切れます。SyncotBase のみと連携していた場合は問題ありませんが、他のサービスにも同じトークンを使っている場合は注意してください。

再連携する場合

LINE連携を再設定するには

解除後、再度 LINE 連携を設定したい場合は以下の手順で行えます。

  1. LINE Developers で Channel Access Token を再発行する

    Step 5 で Revoke した場合は新しいトークンが必要です。「Reissue」ボタンで再発行してください。

  2. LINE Developers で「Use webhook」をオンに戻す

    「Webhook settings」セクションの「Use webhook」トグルをオンにします。

  3. SyncotBase の LINE連携設定に再入力する

    Channel Secret・Channel Access Token・友だち追加URL を再度入力し「保存する」をクリックします。

📄 LINE連携セットアップガイドを参照

初回設定と同じ手順で再連携できます。詳しくは「LINE連携 設定ガイド（LINE_SETUP_GUIDE）」をご参照ください。

FAQ

よくある質問

解除に関してよくお問い合わせいただく内容です。

解除すると患者のデータは消えますか？

いいえ、消えません。予約データ・患者データ・カルテはすべてそのまま保持されます。解除されるのは「Channel Secret」「Channel Access Token」「友だち追加URL」の3つのフィールドのみです。

解除後、患者の LINE アカウントとの友だち関係はどうなりますか？

患者が LINE 公式アカウントを「友だち追加」した状態はそのまま維持されます。LINE 通知が届かなくなるだけで、友だち登録が解除されるわけではありません。

解除後に SyncotBase から LINE 通知を再開したい場合は？

LINE Developers コンソールで Channel Access Token を再発行し、SyncotBase の LINE連携設定ページに再入力するだけで再開できます。詳しくは「再連携する場合」のセクションを参照してください。

「LINE連携を解除する」ボタンが表示されません

このボタンは Channel Secret と Channel Access Token の両方が設定されている場合のみ表示されます。すでにどちらかが空になっている場合（= 実質的に未連携の状態）はボタンが表示されません。その場合、解除の必要はありません。

解除後もメール通知は届きますか？

LINE 解除はメール通知に影響しません。SyncotBase の「通知設定」でメール通知が有効になっていれば、引き続き患者へメールが届きます。解除前にメール通知が有効か確認することをおすすめします。

LINE 側で Webhook を無効化しないといけませんか？

必須ではありません。SyncotBase 側のトークンが削除されるため、LINE から Webhook リクエストが届いても SyncotBase は処理できず、エラーを返します。ただし、セキュリティ面からは LINE Developers で「Use webhook」をオフにしておくことを推奨します。

LINE の Channel 自体を削除したい場合は？

LINE Developers コンソールの該当 Channel の「Basic settings」タブ最下部にある「Delete this channel」から削除できます。ただし 削除は不可逆的です。Channel を削除すると、紐づいた LINE 公式アカウントとの連携も切れるため、慎重に判断してください。通常は Webhook の無効化と Token の Revoke で十分です。


⚠️ LINE 公式アカウント（LINE Official Account Manager 側）は別サービスのため、Channel を削除しても公式アカウントは残ります。
